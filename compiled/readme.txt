1. Для запуска кода запустите файл start.bat

2. В папке "resources" находится конфиг (config.json), 
в котором можно изменить начальные 6 цифр карты (по умол.: 400000) 
и название файла базы (по умол.: banking_system.sqlite). 
В этой-же папке создастся сам файл базы данных sqlite

3. Для проверки числа на соответствие алгоритму Луна,
можно использовать сайт: https://simplycalc.com/luhn-validate.php
